# NLP-APP
Програм хангамжийн хөгжүүлэлт
8 нүүртэй
1. Нэвтрэх / Бүртгүүлэх: Хэрэглэгч и-мэйлээр бүртгүүлж, апп руу нэвтэрнэ.
2. Category сонгох:Зорилгоо дараах үндсэн төрлүүдээс сонгоно. Эрүүл мэнд, хобби, санхүү зэрэг
3. SWOT судалгааны хэсэг: Хэрэглэгч өөрийн зорилготой холбоотой Strengths, Weaknesses, Opportunities, Threats мэдээллээ бөглөнө.
4. Хугацаа, давтамж сонгох: Богино хугацааны эсвэл урт хугацааны гэсэн байдлаар ялгана. Мөн гүйцэтгэх давтамж (өдөр/7 хоног гэх мэт) сонгоно.
5. Ерөнхий төлөвлөгөө гаргах хэсэг: Хэрэглэгчийн бөглөсөн мэдээлэл дээр үндэслэн AI хувийн төлөвлөгөө гаргана.
6. Хэрэглэгчийн Дашбоард: Хийх даалгаврууд, өдөр тутмын төлөвлөгөө, амжилтын хувь, гүйцэтгэлийн график
7. Growth Analyze:Хөгжлийн түвшин, хангалтгүй байгаа хэсгүүд, шинэ төлөвлөгөөний зөвлөмж өгнө.
8. Profile хэсэг: Нэр, нас, хүйс, зорилго, сонгосон категори, төлөвлөгөө, гүйцэтгэл түүх, growth анализын товч тойм, account settings (email update, password reset гэх мэт)
